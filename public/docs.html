GET
/api/hexa/order
Mengeksekusi pembelian instan. Sistem otomatis memotong saldo pembeli, memperbarui stok varian, dan mentransfer laba ke saldo merchant terkait.

Contoh URL Request:

{{config.webUrl}}/api/hexa/order?apikey=API-KEY-KAMU&category=AKUN%20PREMIUM&productId=PRD-147863&subName=1%20BULAN%20FULL&quantity=1

Query Parameters:

ParameterTipeSifatKeterangan
apikeyStringWajibAPI Key akun pembeli.
categoryStringWajibNama kategori induk produk.
productIdStringWajibID produk master target.
subNameStringWajibNama sub-varian produk pilihan.
quantityNumberOpsionalJumlah kuantitas pembelian (Default: 1).

Contoh Response Sukses (JSON):

{
  "success": true,
  "message": "Transaksi sukses terekam dan stok berhasil diperbarui.",
  "dataRahasia": "akun_premium11@gmail.com | pass: dika321",
  "invoiceId": "INV-20260517-991",
  "totalBiaya": 10000,
  "sisaSaldo": 45000
}
GET
/api/hexa/deposit
Membuat tiket tagihan top-up saldo akun otomatis berupa baris kode qris via payment gateway YoBasePay.

Contoh URL Request:

{{config.webUrl}}/api/hexa/deposit?apikey=API-KEY-KAMU&amount=5000

Query Parameters:

ParameterTipeSifatKeterangan
apikeyStringWajibAPI Key pengisi saldo.
amountNumberWajibNominal pengisian saldo (Min: Rp 1.000).

Contoh Response (JSON):

{
  "success": true,
  "data": {
    "trx_id": "TRX-ID",
    "amount": 5000,
    "qrCode": "00020101021226300016COM.MANDIRI...",
    "expiredAt": "2026-05-17T15:15:00Z"
  }
}
GET
/api/hexa/depositstatus
Mengecek riwayat mutasi status QRIS secara *real-time*. Saldo bertambah otomatis jika status dinyatakan lunas (`SUCCESS`).

Contoh URL Request:

{{config.webUrl}}/api/hexa/depositstatus?apikey=API-KEY-KAMU&trxid=TRX-ID

Contoh Response Lunas (JSON):

{
  "success": true,
  "status": "SUCCESS",
  "amount": 5000
}
